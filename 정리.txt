파이썬이 인기가 많은 이유-> 처음 배우기 쉬워서
인공지능, 데이터 분석을 위해서 많이 사용한다. 추가로 게임과 웹 개발 등이 있다.
인터프리터 언어 -> 한줄씩 실행
             ↓ 스크립트 모드
컴파일 언어 -> 통으로 실행
파이썬에서는 " 와 ' 의 구분이 없다.
print -> 출력하는 함수
, -> 뛰어스기
end -> 끝에 붙여준다. 예) print("123", end = "!!")
sap -> 뛰어쓰기 자리에 넣어준다. 예) print("123", "abc", sep = "!!")
input -> 키보드로 값을 입력받는 함수, 반환값은 문자열이다
반환값은 우리가 쓸 수 있는 것
터틀 그래픽(Turtle graphic) -> 거북이를 이용하여 화면에 그림을 그리는 기능
